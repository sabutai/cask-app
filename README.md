# Cask — дневник дегустаций

Cask — твой личный дневник дегустаций виски, вина, пива и других напитков.

## О приложении

Записывай напитки за секунды, веди список желаний, открывай свой вкусовой профиль и следи за статистикой коллекции.

## Возможности

- Дневник дегустаций с оценками и заметками
- Список желаний
- Вкусовой профиль с радар-графиком
- Экспорт в PDF и CSV (премиум)
- Резервное копирование (премиум)
- Умные напоминания (премиум)

## Платформа

Android — доступно в RuStore
